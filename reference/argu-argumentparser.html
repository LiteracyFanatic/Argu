Argu

ArgumentParser

Namespace: Argu
Attributes:
[<AbstractClass>]
[<NoEquality>]
[<NoComparison>]
[<AutoSerializable(false)>]

The Argu type generates an argument parser given a type argument that is an F# discriminated union. It can then be used to parse command line arguments or XML configuration.
